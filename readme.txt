Dialogue is a Common Lisp tool to write dialogs using the Java Standard
Widget Toolkit (SWT) and Foil (a Foreign Object Interface for Lisp).
It provides an abstract layer over the controls and layout capabilities
found in SWT.

Latest version, examples and documentation:

http://www.algo.be/cl/dialogue/index.htm
